El usuario envia una foto o video por telegram --> El bot de telegram pregunta al usuario que quiere hacer (se abre un inlineKeyboard con 3 filtros a aplicar) --> el usuario selecciona filtro --> se manda la opcion seleccionada y el archivo a un server backend levantado con socketserver --> el server con multiprocessing o multithreading aplica a la imagen (o frames si es video) el filtro --> se reconstruye la imagen o video --> se devuelve al bot de telegram --> se devuelve al usuario.
